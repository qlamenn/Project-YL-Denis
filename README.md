# Arcade Defense Game

Игра в стиле "Защита базы" для проекта Яндекс.Лицей.

## Описание
Игрок управляет турелью, защищающей центральное ядро от волн врагов. Цель - продержаться как можно дольше и набрать максимальное количество очков.

## Технологии
- Python 3.x
- Arcade 2.6+
- SQLite (для таблицы рекордов)

## Установка и запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите игру: `python main.py`
